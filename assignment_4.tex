$$ 3d = 1(\text{mod } 40) $$
$$ d = \frac{1(\text{mod } 40)}{3} $$
The nearest integer that gives $ x = 1(\text{mod } 40) $ and is divisible by 3 is 81. Therefore:
$$ d = \frac{81}{3} $$
$$ d = 27 $$

\item For $ m = 8 $, the ciphertext $c$ is:
$$ c = m^e \text{mod } n $$
$$ c = 8^3 \text{mod } 55 $$
$$ c = 512 \text{mod } 55 $$
$$ c = 17 $$
